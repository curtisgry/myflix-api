myFlix is an API for Movie data such as the genre, year released, and director.

Logic URL Method Request data format Response data format
Get list of all movies /movies GET None JSON
Get one movie by title /movies/:title GET None JSON
Get movies by genre /movies/genre/:genreName GET None JSON
Get movies by director /movies/directors/:directorName GET None JSON
Register a new user /users POST JSON JSON
Add a favorite movie for user /users/:userName/favorites POST JSON JSON
Update username /users/:userName POST JSON JSON
Delete a favorite movie /users/:userName/favorites/:favoriteTitle DELETE None None
Delete a user /users/:userName DELETE None None
